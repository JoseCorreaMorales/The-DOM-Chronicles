Estilos HTML - CSS

CSS puede ser agredado a elementos HTML por 3 maneras:

  • En línea -mediante el uso del atributo de estilo en elementos HTML
  • Interno - mediante un elemento style en la sección de la cabezera
  • Externo - mediante un archivo CSS externo

CSS en línea

Se utiliza un CSS en línea para aplicar un estilo único a un solo elemento HTML.

h1 style="color:green"

CSS interno

Un CSS interno es usado para definir un estilo para una sola página HTML.

	<style>
			body {background-color: powderpink;}
			h1   {color: blue;}
			p    {color: lightred;}
	</style>

CSS externo

Se utiliza una hoja de estilo externa para definir el estilo de muchas páginas HTML..

Con una hoja de estilo externa, puede cambiar el aspecto de un sitio web completo, ¡cambiando un archivo!

Para utilizar una hoja de estilo externa, agréguele un vínculo en la sección de la pagina HTML: head

<link rel="stylesheet" href="styles.css">

Fuentes CSS

La propiedad de color CSS define el color del texto que se utilizará.

La propiedad CSS font-family define la fuente que se utilizará.

La propiedad de text-size de CSS define el tamaño del texto que se utilizará.

		h2{
			color:blue;
			font-family: verdana;
			font-size: 300%;
		}
		p{
			color:red;
			font-family: verdana;
			font-size: 300%;
		}

Bordes CSS

La propiedad de borde CSS define un borde alrededor de un elemento HTML

	p{
			border: 1px solid powderblue;
	}

CSS Padding - relleno (espacio)

La propiedad de padding CSS define un relleno (espacio) entre el texto y el borde.

	p {
			border: 1px solid powderblue;
			padding: 30px;
	}

Margen CSS

La propiedad de margin CSS define un margen (espacio) fuera del borde

	p {
			border: 1px solid powderblue;
			margin: 50px;
	}
