HEX value

In HTML, a coloe can be specified using a hexadecimal value in the form

#rrggbb

where rr (red), gg (green) y bb (blue) are values hexadecimales between 00 y ff (same as decimal 0-255)

for example, #ff0000 is displayed as red because is set to the highest value (ff) and the others are set to the lowest value (00)
