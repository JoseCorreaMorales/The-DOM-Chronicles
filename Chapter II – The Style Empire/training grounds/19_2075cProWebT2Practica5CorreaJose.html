El elemento HTML <head>

El elemento <head> es un conterdor para metadatos (datos sobre datos) y se coloca entre la etiqueta <html> y la etiqueta <body>

Los metadatos HTML son datos sobre el documento HTML. No se muestran los metadatos.

Los metadatos suelen definir el título del documento, el conjunto de caracteres, los estilos, los enlaces, los scripts y otra metainformación..

Las siguientes etiquetas describen metadatos: < title >, <style >, < meta>, < link >, < script > y < base >.

El elemento HTML <meta>

El elemento <meta> se utiliza para especificar qué juego de caracteres a utiliza, descripción de la página, palabras clave, autor y otros metadatos..

Los metadatos son utilizados por navegadores (cómo mostrar contenido), por motores de búsqueda (palabras clave) y otros servicios web..

Definir el juego de caracteres a utilizar:

<meta>

definir una descripción de su página web:

<meta name="descripccion content="tutoriales web gratis">

Defina palabras clave para motores de búsqueda:

<meta name="palabras clave" content="HTML CSS, XML, JavaScript">

definir el autor de la pagina

< meta name="autor" content="John Dee">

actualizar el documento cada 30 segundos:

< meta http-equiv="actualizar" content="30">

El Elemento HTML <base>

El elemento<base> especifica la URL base y el destino base para todas las URL relativas en una página:

<base href="C:\wamp64\www\SW\189621ingles\images\"target="_blank">

Omitiendo <html>, <head> y <body>?

Según el estándar HTML5; las etiquetas <html>, la <body>, y la <head> se puede omitir. El siguiente código se validará como HTML5:

<!DOCTYPE html>
				<title> tilulo de la pagina </title>
				<h1>este es un título </h1>
				<p >este es un parrafo </p>

El atributo del juego de caracteres HTML (charset)

Para HTML4:

	 <meta http-equiv="Content-Type" content="text/html; charset=UTF-8/">
	 <meta http-equiv="Content-Type" content="text/html; charset=ISO-8859-1/">

Para HTML5:

< meta charset=UTF-8>
