Instrucciones

Deberá crear los siguientes estilos para poder imitar pagina web

  • Etiquetas <hr> borde inferior con ancho de 10px, solido y color (#ffa20c)
  • Etiquetas en encabezado nivel 2: boede completo con un ancho de 5px, soldido y color (#5eff00)
  • lista <ul> con datos del libro: id "datos" aplicada a la lista con borde superior (border-top: 4px dotted #ff00f2) e inferior con un ancho de 14px, puteado y color rgb(113, 0, 121)
  • Lista <ul> con el contenido del libro: class "contenido" borde (border: 2px dashed rgb(255, 138.0))
  • Etiqueta <img>: con (float: right;) un ancho de 250px, un borde (2px solid rgb(0, 105,42);) y padding y margin de 20px
