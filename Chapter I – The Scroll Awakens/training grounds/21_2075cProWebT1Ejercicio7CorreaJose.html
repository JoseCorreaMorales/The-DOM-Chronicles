Concepts

HTML son las siglas de Lenguaje de marcado de hipertexto, que puede tradicirse como lenguaje de marcas o marcado de hipertexto

el lenguaje HTML5 se emplea para crear las paginas web. Es muy fácil ver el códgo HTML de una pagina web la opción exacata cambia de un navegador a otro y también puede cambiar de una versión a otra del mismo navegador, pero suelen tener un nombre similar.

HTML se compone de etiquetas que se esbriben entre los símbolos "<" y ">"
